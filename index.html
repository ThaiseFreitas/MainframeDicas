🚀 Procedimento de Decolate

1. Solicitação Exemplo

PBFTCC12 0058486 27/05/25
PBFTCC03 0037296 12/07/25

2. Localizar JCL (Tela U)

  • Acesse a TELA U
  • No campo JOB NAME insira o nome solicitado (Ex: PBFTCC12)
  • Ao localizar, entre com "V"
  • Na linha de comando: F JCL
  • Anote o JCL: SYS2.JOBS(FTCJC710)

3. Execução (Tela R)

  • Acesse TELA =R
  • LIB: CTD.V630.REPORTS
  • Comando: F JOB FTCJC710 (JCL localizado anteriormente)
  • Na lista, localize o EVENTUAL e coloque "S"
  • Na tela seguinte, preencha:
    • JOBNAME: Nome solicitado
    • JOBID: J + Jobnumber (Ex: J0021076)
  • Pressione F3, depois F (Force) e Y para confirmar.

4. Validação (Tela A e U)

Confirme a execução na TELA =A. Depois, na TELA U:

  • F4 no relatório
  • Use "S" para conferir filtros (remova se houver)
  • Use MF8 para ir ao final e validar o status: ENDED "OK"

📦 Desmigrar Data Set

Utilizado para Jobs de backup como PTSTIBK1 / PCPBKD19.

Procedimento via P.3.4

  • Identifique o Data Set no ABEND (Ex: NOT SELECTED)
  • Na tela P.3.4, digite o nome do arquivo.
  • Se estiver migrado, digite o comando HRECALL ao lado do arquivo.
  • Aguarde o volume reaparecer e dê RERUN no Job abendado.

Desmigrar via Tela U (Opção 3 - Histórico)

  • Digite "R" (Restore) na frente do Job.
  • Confirme: CONFIRM: Y
  • Restore Mission: RSTEQP01
  • Na TELA =A, localize o RSTEQP01 e dê "B" (Bypass) com Y-Y.

⚠️ Tratamento de Erro (RSTEQP01 Demorando)

  1. Volte na TELA U e dê "C" (Cancel) no Job.
  2. Dê "A" ao lado do Job com problema.
  3. Comando: DI A
  4. Localize e copie o nome do DSNAME (Ex: CTDJB1.JOBSDB...)
  5. Na P.3.4, renomeie o arquivo para .OLD
  6. Na Tela 3, rode o job RSTRESET.
  7. Dê "B" para liberar o RSTEQP01.
